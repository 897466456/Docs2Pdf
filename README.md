# ConvertDoc2Pdf
Convert word/powerpoint/excel file to pdf.
# Supported format
Word: doc, docx
PowerPoint: ppt, pptx
Excel: xls, xlsx
# Usage
Drag a file or a folder into `ConvertDoc2Pdf.vbs`.
